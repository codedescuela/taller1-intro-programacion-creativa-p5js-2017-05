qué entienden por programación?

qué es programar?

  • dar instrucciones a una máquina para que haga lo que tu quieras
    • computador, celular, arduino, tv, microondas, etc.
  • las máquinas no piensan
  • instrucciones precisas, sin ambigüedad
  • instrucciones se entregan en un código (?)

qué es un código?...bueno, esto:

							
								function setup(){
								  createCanvas(200,200);
								}

								function draw(){
								  rect(50,50,20,20);
								}

lo de arriba está escrito en JavaScript!

analicemos el ejemplo anterior:

  • trabajamos sobre una página html
  • creamos un canvas (lienzo) sobre la página html
  • sobre el lienzo dibujamos

la página web se vería así:

  • la página web a través del comando script llama a la librería p5.js
  • luego llama el archivo que contiene el ejemplo: ejemplo_1_1.js
  • al archivo .js se le llama "sketch" o bosquejo

el sketch se vería así:

							
								function setup(){
								  createCanvas(200,200);
								}
								function draw(){
								  rect(50,50,20,20);
								}

  • claro, es lo que vimos antes
  • sobre la página html p5.js crea un lienzo: createCanvas();
  • sobre el canvas podemos dibujar
