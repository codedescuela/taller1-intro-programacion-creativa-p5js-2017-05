Funciones

  • Son secciones de código que ejecutan una tarea específica
  • Algunas funciones requieren argumentos y otras no
  • Hay funciones que retornan valores y otras que no

Funciones: Sintaxis

						
							function nombreFuncion(argumentoFunction){
								instruccion1;
								instruccion2;
								.
								.
								.
								instruccionN;

								return valorRetorno;
							}

  • El comando function declara la función
  • El comando return el resultado de la función (en el caso que sea necesario)

Funciones: Uso y buenas prácticas

  • Usos:
    • Le dan orden y jerarquía al código
    • Permiten trabajar de manera modular
  • Buenas prácticas:
    • Elige nombres simples, como getFecha()
    • Planifica tu código pensando en las funciones, luego diseña cada función

Arreglos: Definición y sintaxis

  • Son listas de variables con un nombre común
  • Permiten trabajar con muchas variables creando sólo una
  • La variable x se declara como arreglo de la siguiente manera:
    • var x = [];
  • Un arreglo está compuesto de elementos, y cada elemento tiene in índice (0,1,2...)
  • El primer elemento siempre tiene el número 0

Arreglos: Noche estrellada (Ejemplo 4.2)

						
							//Arreglos
							var x = [];
							var y = [];
							var diam = [];
							var numEstrellas = 100;

							function setup(){
							  createCanvas(800,500);
							  background(30);
							  for (var i = 0; i < numEstrellas; i++) {
							    x[i] = random(0,width);
							    y[i] = random(0,height);
							    diam[i] = random(1,7);
							  }

							}

							function draw(){
							  for (var i = 0; i < numEstrellas; i++) {
							    noStroke();
							    fill(240);
							    ellipse(x[i],y[i],diam[i],diam[i]);
							  }
							}

  • Creamos una noche estrellada usando for y arreglos

Arreglos: Seguidor del cursor (Ejemplo 4.3)

							
								var num = 30;
								var x = [];
								var y = [];

								function setup(){
								  createCanvas(800,500);
								  noStroke();

								  for (var i = 0; i < num; i++) {
								    x[i] = 0;
								    y[i] = 0;
								  }
								}

								function draw(){
								  background(0);
								  frameRate(100);
								  for (var i = num-1; i >0 ; i--) {
								    x[i] = x[i-1];
								    y[i] = y[i-1];
								  }
								  x[0] = mouseX;
								  y[0] = mouseY;

								  for (var i = 0; i < num; i++) {
								    fill(255 - i*4);
								    ellipse(x[i], y[i], 40, 40);
								  }
								}

Trabajar con imágenes en p5.js

  • p5.js tiene una serie de funciones que permiten cargar imágenes en un sketch con siguientes pasos:
      • 1. Crear una variable que aloje la imagen
      2. Usar preload() antes de setup() para cargar la imagen antes de comenzar el programa
      3. En de preload() asignamos la imagen a la variable usando loadImage()
      4. En draw() desplegamos la imagen usando image()

Trabajar con imágenes en p5.js (4.4)

							
								var foto;

								function preload(){
								  foto = loadImage("coded.png");
								}

								function setup(){
								  createCanvas(500,500);
								}

								function draw(){
								  image(foto,0,0);
								}
