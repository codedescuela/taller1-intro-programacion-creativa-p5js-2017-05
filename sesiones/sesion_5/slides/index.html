Objetos

  • Son colecciones de variables (propiedades) y funciones (métodos) que se relacionan entre ellas
  • Permiten representar abstracciones del mundo real

Objetos

Por ejemplo una radio:

  • Propiedades
    • volumen, frecuencia, banda (AM, FM), encendido/apagado
  • Métodos
    • fijarVolumen, fijarFrecuencia, fijarBanda, encenderApagar

Objetos: Función constructora

  • Para declarar un objeto primero debemos definir el Constructor
  • Es una función que define la estructura del objeto
  • Es como el plano de una casa: se pueden construir muchas casas con un plano pero no todas son idénticas

Objetos (5.1)

						
							//Ejemplo 5.1 Objetos
							//Coded Escuela - 2017

							var x;
							var y;
							var rad;
							var colorCirc;
							var vel;

							function setup(){
							  createCanvas(800,500);

							  x = width/2;
							  y = height/2;
							  rad = 30;
							  vel = 2;

							  colorCirc = color(random(0,255), random(0,255), random(0,255), 180);
							}

							function draw(){
							  frameRate(45);
							  background(220,220,255);

							  update();
							  mover();

							}

							function update(){
							  noStroke();
							  fill(colorCirc);
							  ellipse(x,y,2*rad,2*rad);
							}

							function mover(){
							  x += random(-vel, vel);
							  y += random(-vel, vel);
							}

Objetos: de Círculo a Objeto Círculo

  • La círculo tiene ciertas propiedades que lo caracterizan:
    • x, y, rad, vel, colorCirc
  • y ciertos métodos que generan su movimiento
    • function update()
    • function mover()

De Círculo a Objeto Círculo: Constructor

						
							function Circulo(){
							  //Propiedades

							  //Metodos

							}

  • El constructor Circulo() contiene todas las propiedades y metodos del objeto Circulo
  • Se define al final del código, fuera de la función draw()

De Círculo a Objeto Círculo: Propiedades

						
							function Circulo(x,y,radio,colorInterior,velocidad) {
							  //Propiedades
							  this.x = x;
							  this.y = y;
							  this.rad = radio;
							  this.colorCirc = colorInterior;
							  this.vel = velocidad;

							  //Metodos

							}

  • Define las propiedades usando this.varName
  • Asigna los valores con una variable de paso: this.rad = radio;

De Círculo a Objeto Círculo: Métodos

						
							function Circulo(x,y,radio,colorInterior,velocidad) {
							  //Propiedades
							  this.x = x;
							  this.y = y;
							  this.rad = radio;
							  this.colorCirc = colorInterior;
							  this.vel = velocidad;

							  //Metodos
							  this.update = function(){
							    noStroke();
							    fill(this.colorCirc);
							    ellipse(this.x,this.y,2*this.rad,2*this.rad);
							  }

							  this.mover = function(){
							    this.x += random(-this.vel, this.vel);
							    this.y += random(-this.vel, this.vel);
							  }

							}

  • Declara los métodos como funciones

De Círculo a Objeto Círculo: Nuevo Objeto

  • Para crear un nuevo círculo debemos declarar una variable que contenga la nueva instancia del Objeto Círculo
    • var circulo;
  • Creamos el nuevo círculo usando la palabra clave new y entregando los argumentos definidos en el constructor
    • circulo = new Circulo();

Arreglos de Objetos

  • Podemos utilizar un solo constructor para crear muchos Objetos similares
  • Por ejemplo, muchos círculos de distinos tamaños y colores que se muevan a velocidades similares

Arreglos de Objetos

  • En vez de usar una variable que contenga el Objeto, utilizaremos un arreglo que contenga muchos objetos
  • Para llenar el arreglo con los objetos y ejecutar los métodos de cada círculo debemos usar un for()

Arreglo de Objetos (5.3)

						
							//Ejemplo 5.3 Objetos
							//Coded Escuela - 2017

							//Circulo
							var circulos = [];

							//Numero circulos
							numCirc = 80;

							function setup(){
							  createCanvas(800,500);

							  for (var i = 0; i < numCirc; i++) {
							    circulos[i] = new Circulo(random(0,width),random(0,height),random(20,30),color(random(0,255), random(0,255), random(0,255), 180),2);
							  }
							}

							function draw(){
							  frameRate(45);
							  background(220,220,255);

							  for (var i = 0; i < numCirc; i++) {
							    circulos[i].update();
							    circulos[i].mover();
							  }

							}

							function Circulo(x,y,rad,colorCirc,vel) {
							  //Propiedades
							  this.x = x;
							  this.y = y;
							  this.rad = rad;
							  this.colorCirc = colorCirc;
							  this.vel = vel;

							  //Metodos
							  this.update = function(){
							    noStroke();
							    fill(this.colorCirc);
							    ellipse(this.x,this.y,2*this.rad,2*this.rad);
							  }

							  this.mover = function(){
							    this.x += random(-this.vel, this.vel);
							    this.y += random(-this.vel, this.vel);

							  }

							}
