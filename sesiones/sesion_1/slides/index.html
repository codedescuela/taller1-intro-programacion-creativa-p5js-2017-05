La página web se vería así:

  • La página web a través del comando script llama a la biblioteca p5.js
  • Luego llama el archivo que contiene el ejemplo: ejemplo_1_1.js
  • al archivo .js se le llama "sketch" o bosquejo

El sketch se vería así:

							
								function setup(){
								  createCanvas(200,200);
								}
								function draw(){
								  rect(100,100,50,50);
								}

  • claro, es lo que vimos antes
  • sobre la página HTML p5.js crea un lienzo: createCanvas();
  • sobre el canvas podemos dibujar

orden de ejecución del código:

							
								function setup(){
								  createCanvas(200,200);
								}
								function draw(){
								  rect(100,100,50,50);
								}

  • setup() y draw() son funciones. cumplen una tarea específica
  • la máquina lee el código de izquierda a derecha y de arriba hacia abajo, como nosotros

orden de ejecución del código: función setup()

							
								function setup(){
								  createCanvas(200,200);
								}

  • la función setup() define lo mínimo que la máquina necesita para ejecutar el código
  • en este caso "declara" la creación del canvas (lienzo) a través de createCanvas()

Orden de ejecución del código: función draw()

							
								function draw(){
								  rect(100,100,50,50);
								}

  • La función draw()dice que todo lo que está entre llaves {} se dibuja
  • en este caso dibujamos un cuadrado
  • después explicaremos lo que está en el paréntesis del rect(50,50,20,20);

Sintaxis: dos cosas importantes al respecto

							
								function setup(){
								  createCanvas(200,200);
								}
								function draw(){
								  rect(100,100,50,50);
								}

  • Bloque de código: es todo lo que está dentro llaves{}
  • punto y coma (;):
    • Imprescindibles en javaScript
    • indican el fin de cada línea de código

funciones (1/2): sintáxis

  • "Una función es un conjunto de líneas de código que realizan una tarea específica y puede retornar un valor. Las funciones pueden tomar parámetros que modifiquen su funcionamiento." Fuente
  • sintáxis: funcion();
  • cada función tiene una sintáxis.
    • Referencia de p5.js

funciones (1/2): tipo

  • argumentos: funciones que requieren argumento y funciones que no
  • retornos: funciones que retornan un valor y funciones que no
    • este tipo lo veremos más adelante

funciones (1/2): argumentos

  • funciones que requieren argumento:
    • createCanvas(300,200)
    • ellipse(100,100,50,50)
  • funciones que no requieren argumento:
    • setup()
    • draw()

funciones (1/2): createCanvas()

  • crea un lienzo (canvas) sobre el cual dibujar, con parámetros:
    • ancho (width)
    • alto (height)

funciones (1/2): createCanvas()

  • mostrar un esquema de un canvas
  • un lienzo es una gran cantidad de puntos o pixeles
  • se ordenan como una matriz, en filas y columnas
  • cada pixel corresponde a un par ordenado (x,y)

funciones (1/2): volvamos al ejemplo 1.2

  • el lienzo creado no tiene color

funciones (1/2): color del lienzo (1.3)

						
							function setup(){
							  createCanvas(300,200);
							  background(150);
							}

							function draw(){
							  ellipse(150,100,90,90);
							}

  • background(150) hace que el linzo sea gris
  • ahora lo podemos ver!
  • qué pasa si probamos con otros números?
