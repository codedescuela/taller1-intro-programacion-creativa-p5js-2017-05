Ciclos for (for loops): Definición

  • Para hacer este tipo de operaciones más eficientes, en JavasCript existen los ciclos
  • Los ciclos ejecutan un bloque de código la cantidad de veces que se desee
  • Hay tres tipo de ciclos:
    • for()
    • for in()
    • while()
  • Por ahora solo aprenderemos el for()

Ciclos for (for loops): Sintaxis

  • Para crear un ciclo for() se requieren tres elementos:
    • Inicialización de contador
    • Testeo de condición de ciclo
    • Incremento de contador

Ciclos for (for loops): Sintaxis

						
							for (var i = 0; i < 8; i++) {
							  //código
							}

  • La variable i se donimina contador y se inicializa (en este ejemplo) con el valor 0
  • Se evalúa si el contador i es menor que 8, mientras esto se cumpla el for() se seguirá ejecutando
  • El contador i se incrementa en 1 a cada ciclo
  • Cuando el contador i deja de ser menor que 8, se termina el for()
