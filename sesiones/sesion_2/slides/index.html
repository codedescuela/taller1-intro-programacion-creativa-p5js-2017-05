Variables en JavaScript

  • Una variable es una porción de memoria del computador que almacena información
  • Una variable puede almacenar distitnos tipos de información:
    • Números enteros (int)
    • Números decimales (float)
    • Texto (string)
  • La información almacenada por una variable puede ser modificada muchas veces

Variables en JavaScript: Declarar variable

  • Debes elegir un nombre para la variable
  • El nombre debe representar la información contenida
  • Si la variable es el radio de un círculo es mejor llamarla radio en vez de r
  • Hay nombres que JavaScript y p5.js reservan para sus variables internas

Variables en JavaScript: Declarar variable (2.1)

							
								function setup(){
								  createCanvas(480,120);
								}

								function draw(){
								  background(204);
								  rect(150,150,90,90);
								  ellipse(75, 100, 130, 130);
								  ellipse(175, 100, 130, 130);
								  ellipse(275, 100, 130, 130);
								}

  • El ejemplo 2.1 muestra tres círculos traslapados
  • El código es ineficiente. Hay valores que se repiten
  • Para no repetir información podemos usar variables

Variables en JavaScript: Declarar variable

  • Hay dos etapas en la creación de una variable: declarar y asignar valor
  • El nombre de la variable debe ser informativo, supongamos y para la posición y del círculo
  • Declaración: var y;

							
								var y;		//Declaración
								y = 100;	//Asignación de valor

Variables en JavaScript: Declarar variable

  • Al declarar la variable le ordenamos al computador que debe crear una variable con el nombre y
  • Al asignar el valor 100 le indicamos que la variable y debe almacenar el número 100
  • Ambos procesos se pueden hacer en un sólo paso

							
								var y = 100;	//Declaración y asignación de valor

Variables en JavaScript: Buenas prácticas

  • Existen variables globales y variables locales. La diferencia la veremos más adelante
  • Siempre elije nombres cortos pero que definan claramente el contenido de la variable
  • Utiliza el formato serpiente-camello
    • var radCirculo;
  • No dupliques nombres de variables

Variables en JavaScript: Uso de las variables

  • Utilizamos la variable y para reemplazar el valor 100 en las tres elipses
  • Beneficio de usar la variable y:
    • Si quermeos modificar la posición y de las tres elipses sólo debemos modificar el valor de la variable y
    • Hace más ordenado nuestro código
  • También podemos crear la variable d para el diámtro de la elipse
