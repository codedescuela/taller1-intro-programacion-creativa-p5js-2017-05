Variables en JavaScript

  • Una variable es una porción de memoria del computador que almacena información
  • Una variable puede almacenar distitnos tipos de información:
    • Números enteros (int)
    • Números decimales (float)
    • Texto (string)
  • La información almacenada por una variable puede ser modificada muchas veces

Variables en JavaScript: Declarar variable

  • Debes elegir un nombre para la variable
  • El nombre debe representar la información contenida
  • Si la variable es el radio de un círculo es mejor llamarla radio en vez de r
  • Hay nombres que JavaScript y p5.js reservan para sus variables internas

Variables en JavaScript: Declarar variable (2.1)

							
								function setup(){
								  createCanvas(480,120);
								}

								function draw(){
								  background(204);
								  rect(150,150,90,90);
								  ellipse(75, 100, 130, 130);
								  ellipse(175, 100, 130, 130);
								  ellipse(275, 100, 130, 130);
								}

  • El ejemplo 2.1 muestra tres círculos traslapados
  • El código es ineficiente. Hay valores que se repiten
  • Para no repetir información podemos usar variables

Variables en JavaScript: Declarar variable

  • Hay dos etapas en la creación de una variable: declarar y asignar valor
  • El nombre de la variable debe ser informativo, supongamos y para la posición y del círculo
  • Declaración: var y;

							
								var y;		//Declaración
								y = 100;	//Asignación de valor

Variables en JavaScript: Declarar variable

  • Al declarar la variable le ordenamos al computador que debe crear una variable con el nombre y
  • Al asignar el valor 100 le indicamos que la variable y debe almacenar el número 100
  • Ambos procesos se pueden hacer en un sólo paso

							
								var y = 100;	//Declaración y asignación de valor

Variables en JavaScript: Buenas prácticas

  • Existen variables globales y variables locales. La diferencia la veremos más adelante
  • Siempre elije nombres cortos pero que definan claramente el contenido de la variable
  • Utiliza el formato serpiente-camello
    • var radCirculo;
  • No dupliques nombres de variables

Variables en JavaScript: Uso de las variables

  • Utilizamos la variable y para reemplazar el valor 100 en las tres elipses
  • Beneficio de usar la variable y:
    • Si quermeos modificar la posición y de las tres elipses sólo debemos modificar el valor de la variable y
    • Hace más ordenado nuestro código
  • También podemos crear la variable d para el diámtro de la elipse

Operaciones matemáticas

  • En el ejemplo anterior sumamos valores
  • En programación, las operaciones matemáticas se representan con los siguientes símbolos:
    • + para sumar dos valores
    • - para restar dos valores
    • * para multiplicar dos valores
    • / para dividir dos valores
    • = para asignar un valor a una variable

Operadores matemáticos

  • En JavaScript existen ciertos operadores matemáticos que resumen operaciones:
    • "x += 10" representa "x = x + 10"
    • "x -= 51" representa "x = x - 51"
  • Existen casos especiales para el número 1:
    • "x ++" representa "x = x + 1"
    • "x --" representa "x = x - 1"

Aritmética básica: Orden de procedencia

  • En programación existen reglas tomadas de la matemática para ordenar el cálculo de las operaciones
  • En el siguiente código:

						
							var gradCels;
							var gradFahr = 50;

							gradCels = 5/9 * (gradFahr - 32);

Aritmética básica: Orden de procedencia

  • El orden de procedencia indica que la prioridad de los operadores es el siguiente:
    • Primero, lo que está entre paréntesis (sean curvos, llaves o corchetes)
    • Segundo, las multiplicaciones y divisiones
    • Tercero, las sumas y restas
    • Cuarto, cuando hay órdenes de procedencia iguales se calcula de izquierda a derecha

Aritmética básica: Orden de procedencia

						
							var gradCels;
							var gradFahr = 50;

							gradCels = 5/9 * (gradFahr - 32);

  • Luego en el ejemplo el cálculo se realiza en el siguiente orden:
    • Primero, se calcula (gradFahr - 32)
    • Segundo, se calcula 5/9
    • Tercero, se calcula la multiplicación entre los resultados anteriores

Color: RGB

  • Para representar el color utilizamos el modelo RGB
  • En el modelo RGB cada color se construye a partir de la suma de 3 colores básicos:
    • Rojo (Red)
    • Verde (Green)
    • Azul (Blue)
  • Cada color puede tener un valor entre 0 - 255
  • La ausencia de color (R = 0, G = 0, B = 0) representa el negro
  • La presencia total de color (R = 255, G = 255, B = 255) representa el negro

Color: RGB (2.6)

							function setup(){
							  createCanvas(800,500);
							}

							function draw(){
							  background(230,230,255);
							  fill(210,0,0);
							  ellipse(mouseX, mouseY, d ,d);
							}
  • En el Ejmplo 2.4 podemos pintar la elipse de un color rojo y el fondo de un color azul claro
  • La función fill() rellena el interior de la elipse

Contornos: función stroke()

  • Stroke en inglés significa trazo, y en p5.js se usa para hablar del contorno de una figura
  • Las funciones de contorno se deben escribir antes de la función que dibuja la figura:
    • noStroke(): elimina el contorno de la figura
    • stroke(color): colorea el contorno de la figura con el color indicado en RGB
    • strokeWeight(): define el ancho del contorno

Contornos: función stroke() (2.8)

						
							var d = 130;

							function setup(){
							  createCanvas(800,500);
							}

							function draw(){
							  background(230,230,255);
							  strokeWeight(6);
							  stroke(130,0,0);
							  fill(210,0,0);
							  ellipse(mouseX, mouseY, d ,d);
							}
