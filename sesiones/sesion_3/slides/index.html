Ciclos for (for loops): Definición

  • Para hacer este tipo de operaciones más eficientes, en JavasCript existen los ciclos
  • Los ciclos ejecutan un bloque de código la cantidad de veces que se desee
  • Hay tres tipo de ciclos:
    • for()
    • for in()
    • while()
  • Por ahora solo aprenderemos el for()

Ciclos for (for loops): Sintaxis

  • Para crear un ciclo for() se requieren tres elementos:
    • Inicialización de contador
    • Testeo de condición de ciclo
    • Incremento de contador

Ciclos for (for loops): Sintaxis

						
							for (var i = 0; i < 7; i++) {
							  //código
							}

  • La variable i se donimina contador y se inicializa (en este ejemplo) con el valor 0
  • Se evalúa si el contador i es menor que 7, mientras esto se cumpla el for() se seguirá ejecutando
  • El contador i se incrementa en cada ciclo
  • Cuando el contador i deja de ser menor que 7, se termina el for()

Ciclos for (for loops) (3.2)

						
							var d = 130;

							function setup(){
							  createCanvas(480,240);
							}

							function draw(){
							  background(204);
							  for (var i = 0; i < 7; i++) {
							    ellipse(65*(1+i), height/2, d, d);
							  }
							}

Ciclos for anidados y color aleatorio

  • Un for() anidado permite recorrer dos dimensiones y construir una matriz de elipses
  • Agreguems color a las elipses y hagamos que cambie aleatoriamente, para ello usaremos dos nuevas funciones:
    • random(a,b) entrega un valor aleatorio equiprobable entre el valor a y el valor b
    • frameRate() define cuántas veces se ejecuta la función draw() en un segundo

Ciclos for anidados (3.3)

						
							var d = 90;     //Diametro de los circulos
							var numX = 8;   //Cantidad de circulos en el eje horizontal
							var numY = 5;   //Cantidad de circulos en el eje vertical
							var red;        //Variable asociada al color rojo
							var green;      //Variable asociada al color verde
							var blue;       //Variable asociada al color azul

							function setup(){
							  createCanvas(800,500);
							}

							function draw(){
							  frameRate(7);
							  background(100);
							  for (var i = 0; i <= numX; i++) {
							    for (var j = 0; j <= numY; j++) {
							      red = random(0,255);
							      green = random(0,255);
							      blue = random(0,255);

							      fill(red, green, blue, 150);
							      ellipse(i*width/numX, j*height/numY, d, d);
							    }
							  }
							}

Condicional if

  • Se utiliza para condicionar una acción al resultado de una prueba lógica (variables booleanas pueden tomar un valor true o false)
  • Por ejemplo, siendo a y b variables:
    • Si a es mayor que b realizar cierta acción
    • Si a es igual a b realizar cierta acción

Condicional if: Sintaxis

						
							if(pruebaLogica == true){
								//Ejecutar Código
							}

Condicional if: Operadores relacionales

  • En JavaScript existen operadores que permiten evaluar la relación entre dos parámatros:
    • a > b, implica que a es mayor que b
    • a < b, implica que a es menor que b
    • a >= b, implica que a es mayor igual a b
    • a <= b, implica que a es menor igual a b
    • a == b, implica que a es igual a b
    • a != b, implica que a es distinto de b

Condicional if (3.5)

						
							var x;
							var offset = 10;

							function setup(){
							  createCanvas(800,500);
							  x = width/2
							}

							function draw(){
							  background(204);
							  if (mouseX > x) {
							    x += 2;
							    offset = -10;
							  }
							  if (mouseX < x) {
							    x -= 2;
							    offset = 10;
							  }
							  //Dibujar una flecha hacia la izquierda o derecha dependiendo del offset
							  line(x, 0, x, height);
							  line(mouseX, mouseY, mouseX + offset, mouseY - 10);
							  line(mouseX, mouseY, mouseX + offset, mouseY + 10);
							  line(mouseX, mouseY, mouseX + offset * 3, mouseY);
							}

Condicional if / else if() / else

  • Para los casos en que queremos evaluar más de una prueba lógica se puede utilizar los condicionales else-if y else
    • else if()
      • se utiliza para evaluar una prueba lógica secundaria
    • else
      • se utiliza para evaluar las pruebas lógicas restantes

Condicional if / else if() / else: Sintaxis

						
							if(pruebaLogica1){
								//Ejecutar Codigo Prueba Logica 1
							}
							else if(pruebaLogica2){
								//Ejecutar Codigo Prueba Logica 2
							}
							else if(pruebaLogica3){
								//Ejecutar Codigo Prueba Logica 3
							}
							else {
								//Ejecutar Codigo
							}

  • Se puede utilizar cuantos else if() sean necesarios

Condicional if / else if() / else (3.6)

						
							var diametro = 150;
							var distancia = 0;
							var colorCirculo;

							function setup(){
							  createCanvas(800,500);
							  background(255,255,200);
							}

							function draw(){
							  distancia = dist(mouseX,mouseY,width/2,height/2);

							    colorCirculo = color(150,0,0);
							  }
							  else {

							  }

							  strokeWeight(4);
							  fill(colorCirculo);
							  ellipse(width/2,height/2,diametro,diametro);
							}

  • dist() calcula la distancia entre dos puntos

Operadores lógicos: Definición y sintaxis

  • En casos en que la prueba lógica se compone de dos o más pruebas se deben usar los operadores lógicos
  • Se utilizan para relacionar dos instancias booleanas true o false
    • AND (y): &&
    • OR (o): ||

Operadores lógicos

  • AND:
    • true && true = true
    • true && false = false
    • false && false = false
  • OR:
    • true || true = true
    • true || false = true
    • false || false = false
